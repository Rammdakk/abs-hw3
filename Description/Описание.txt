Базовые альтернативы (уникальные параметры, задающие отличительные признаки альтернатив):
1. Шифрование заменой символов (массив пар: [текущий символ, замещающий символ]; зашифрованный текст – строка символов) -  symbols
2. Шифрование цикличе- ским сдвигом кода каждого символа на n (целое число, определяющее сдвиг; за- шифрованный текст – строка символов) - cyclic
3. Шифрование заменой символов на числа (пары: текущий символ, целое число – подстановка при шифровании кода символа в виде короткого целого; зашифрованный текст – целочисленный массив) - number
Общие для всех альтернатив переменные:
Открытый текст – строка символов.
Общие для всех альтернатив функции:
Частное от деления суммы кодов незашифрованной строки на число символов в этой строке (действительное число).

Вод данных:
С файла:
Замена символов - 1,
Далее вводится зашифрованная строка, а затем символ - шифр и символ -соответствие для каждого символа строки. 
Циклическое шифрование - 2,
Далее вводится зашифрованная строка, а затем число - число сдвига кода символов. 
Шифрование заменой на числа - 3
Вводится длина строки, затем последовательность чисел, которые являются зашифрованными символами, затем код шифрования - символ, а затем соответсвующее ему число в шифре. 
Длина строк ограничена 100.
